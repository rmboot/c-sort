# c-sort

| :------: | :------: | :------: | :------: | :-----:| :------: | :--: |
| 交 | 冒泡排序 | O(n^2) | O(n^2) | 稳定| O(1) | n小时较好 |

| 选 | 选择排序 | O(n^2) | O(n^2) | 不稳定| O(1) | n小时较好 |
| 择 | 堆排序 | O(nlogn) | O(nlogn) | 不稳定| O(1) | n大时较好 |

| 插 | 插入排序 | O(n^2) | O(n^2) | 稳定| O(1) | 大部分已排序时较好 |
